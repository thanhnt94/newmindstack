Khôi phục từ file tải lên

Chọn file sao lưu (.json hoặc .zip) đã xuất từ hệ thống để khôi phục dữ liệu tương ứng.

Hỗ trợ gói dữ liệu nội dung, bản sao lưu cơ sở dữ liệu và gói sao lưu toàn bộ.

Hệ thống sẽ tự động nhận diện loại gói sao lưu và ghi đè dữ liệu tương ứng.

  • Gói dữ liệu nội dung: ghi đè các bảng dữ liệu liên quan.
  • Bản sao lưu cơ sở dữ liệu/toàn bộ: ghi đè file cơ sở dữ liệu và thư mục uploads (nếu có).

Dữ liệu hiện tại sẽ được thay thế bằng nội dung trong file đã tải lên.
